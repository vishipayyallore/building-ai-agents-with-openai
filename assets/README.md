# assets/

Shared course/media for the Engineering Organization and Teaching Product projections.

Prefer:

```text
assets/images/     # shared diagrams and screenshots
assets/            # other shared types as needed (gif/, sample-data/, …)
```

Session-specific media prefers `sessions/session-NN-*/assets/`.

Replaces a root `images/` tree ([ADR-008](../docs/ADRs/ADR-008-product-organization-publishing.md)).
