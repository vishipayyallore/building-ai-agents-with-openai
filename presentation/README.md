## Workflow

**Session 1 note:** Demo 1 is tagged at `v1.0-build-your-first-agent`. Attendees checkout that tag and follow [docs/03-getting-started.md](../docs/03-getting-started.md). From Session 2 onward, create both a start tag and a completion tag.

1. Complete the session on `main`.
2. Update the relevant demo README, release-notes file, and supporting docs.
3. Create the annotated tag for the completed milestone.
4. Create the GitHub Release using the same tag name.
5. Link the release notes back to the demo README and canonical docs.
